Project 30) Iron walkers Mark II (150 copies) 
(2025)
This new version of Iron walkers has a slight changes, instead of using D-walker's weapons, 
I decided to combine the design with Ahab's chopper's (from metal gear solid 5 phantom pain) Ahab's chopper has interesting features that I added in this version such as camouflages
Flares, different weapon systems And shields,
I also added the chopper's blades to the left arm of the armor to use it as a shield blade that work with high speed to defend both the user and the armor, 
The flight and rider system is still intact, and the body is still a full tactical titanium armor,
